# 💇‍♀️ Glow Beauty Salon Website

## 📌 Project Overview
This project is a professional website created for a sample local business called **Glow Beauty Salon**.  
The goal of this project is to provide an online presence for small businesses and help them attract more customers.

---

## 🎯 Objective
To design and develop a simple, responsive, and user-friendly website for a local business using basic web development technologies.

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

---

## 🌐 Features
- Clean and simple user interface  
- About section describing the business  
- Services section listing available services  
- Contact information for easy communication  
- Basic responsive layout  

---

## 🚀 How This Website Helps the Business
This website helps the business in the following ways:

- Improves online visibility
- Helps customers easily explore services  
- Builds trust with a professional look  
- Makes it easier for customers to contact the business

---

---

## ▶️ How to Run the Project
1. Download or clone the repository  
2. Open the folder in VS Code  
3. Open `index.html` in a browser  
   or use **Live Server**
